# Flutter Inquiry Form

A Flutter application demonstrating a **reusable custom button widget** and an **Inquiry / Contact form** integrated with **Firebase Firestore**.

This project was created as part of a Flutter assignment to demonstrate reusable widget design and backend integration.

---

## Features

- Reusable Custom Button Widget
- Loading state support
- Custom styling support
- Inquiry / Contact Form UI
- Firebase Firestore integration
- Clean project structure
- Follows modular Flutter architecture

---

## Tech Stack

- **Flutter**
- **Dart**
- **Firebase**
- **Cloud Firestore**
