# Flutter Reusable Button Widget

## Overview
This project implements a reusable custom button component in Flutter.

## Features
- Loading state support
- Custom styles (color, size, text)
- Clean architecture
- Follows SOLID principles

## Usage Example
ReusableButton(
  text: "Submit",
  isLoading: false,
  onPressed: () {
    print("Button Pressed");
  },
)

## Tech Stack
- Flutter
- Dart
